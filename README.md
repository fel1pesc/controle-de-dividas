O desafio foi feito na linguagem PHP, com o framework Laravel.
Para executa-lo, abra o prompt de comando, vá até o diretório do projeto e execute o comando que está abaixo:
- composer update

Após executado, execute o comando para rodar o servidor em sua máquina local:
- php artisan serve

Depois de ser executado, acesse:
- http://localhost:8000
